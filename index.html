Sistem Penerimaan Mahasiswa Baru Terintegrasi

Ringkasan Proyek

Sistem informasi web berbasis Spring Boot Framework untuk mengelola proses penerimaan mahasiswa baru (PMB) di Universitas HKBP Nommensen secara terintegrasi, otomatis, dan transparan.

Nama Penulis: Mychael Daniel N (122140104) | Program: Teknik Informatika | Institusi: Institut Teknologi Sumatera

Technology Stack

Java 21 Spring Boot 3.2 Spring Security Spring Data JPA MySQL 8.0 JWT Authentication Gmail SMTP BRIVA API Bootstrap 5 HTML5/CSS3/JavaScript Lombok Maven

Fitur yang Telah Diimplementasikan

Core Backend

  • Spring Boot Application dengan konfigurasi lengkap
  • 9 Entity Models (User, Student, RegistrationPeriod, SelectionType, AdmissionForm, VirtualAccount, Exam, ExamResult, ReEnrollment, Notification, SystemConfiguration)
  • 11 Repository Interfaces untuk data access
  • Layanan bisnis (AuthService, StudentRegistrationService, BrivaService, EmailService)
  • JWT Token Authentication & Authorization
  • Spring Security Configuration dengan role-based access control
  • Exception Handling & Logging

REST APIs

  • Authentication APIs (Register, Login, Validate Token)
  • Student (CAMABA) APIs (Profile, Registration, Exam, Status)
  • Admin APIs (Period Management, Validation, Results Publishing)
  • Payment Integration APIs (Virtual Account, Verification)
  • Email Notification APIs

Database Schema

  • 12 Database Tables dengan relasi yang tepat
  • Primary & Foreign Keys
  • Timestamps (created_at, updated_at)
  • Enums untuk status management

Frontend Pages

  • Landing Page dengan fitur dan alur pendaftaran
  • Login Page dengan responsive design
  • Registration Page dengan password strength indicator
  • Bootstrap 5 integration untuk responsive design
  • JavaScript untuk interaksi dan validasi client-side

External Integrations

  • Gmail SMTP untuk email notifications
  • BRIVA Payment Gateway API untuk Virtual Account generation
  • Google Forms untuk ujian online

Security Features

  • JWT Token-based Authentication
  • Role-Based Access Control (CAMABA, ADMIN_VALIDASI, ADMIN_PUSAT)
  • Password Encryption (BCrypt)
  • CORS Configuration
  • Stateless Session Management

Detail Implementasi

Database Setup (application.properties)

spring.datasource.url=jdbc:mysql://localhost:3306/pmb_uhn spring.datasource.username=root spring.datasource.password= spring.jpa.hibernate.ddl-auto=update spring.jpa.properties.hibernate.dialect=org.hibernate.dialect.MySQL8Dialect

Email Configuration

spring.mail.host=smtp.gmail.com spring.mail.port=587 spring.mail.username=${MAIL_USERNAME} spring.mail.password=${MAIL_PASSWORD} spring.mail.properties.mail.smtp.auth=true spring.mail.properties.mail.smtp.starttls.enable=true

JWT Configuration

jwt.secret=${JWT_SECRET} jwt.expiration=86400000 # 24 hours

API Endpoints

Method Endpoint Deskripsi Role
POST /api/auth/register Register akun baru Public
POST /api/auth/login Login ke sistem Public
GET /api/camaba/profile Get student profile CAMABA
PUT /api/camaba/profile Update student profile CAMABA
POST /api/camaba/register-admission Register for admission CAMABA
POST /api/camaba/buy-form/{formId} Buy form & create VA CAMABA
GET /api/admin/periods Get all periods ADMIN
POST /api/admin/periods Create period ADMIN_PUSAT

Instruksi Setup

1. Persiapan

  • Install Java JDK 21
  • Install Maven 3.8+
  • Install MySQL 8.0+

2. Database

CREATE DATABASE pmb_uhn CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

3. Build & Run

mvn clean install mvn spring-boot:run

4. Access Application

Buka browser dan akses: http://localhost:8080
